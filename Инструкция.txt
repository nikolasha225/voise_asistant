Инструкция к ассистенту:
	Для работы ассистента требуется:
		Установленный python3.10 и новее(В архиве есть установщик 3.11)
		Выход к интернету
	Первый запуск:
		Запустите Settings.exe
		Открылась командная строка, в ней будет выполняться настройка
		!При нажати функциональных клавиш(F1,F2,F3...) следует сначала кликнуть на пустой рабочий стол, т.к. они запускают некоторые функции консоли(например f11 делайт полноэкранный режим и обратно)!
		Нажмите f2
			Настрока запуска приложений
				Функция запуска приложения работает так:
					Вы говорите реплику запуска и реплику приложения
					Ассистент отвечает
					Запускается приложение(возможно при пролаге системы откроетя пустая консоль, её можно закрыть)
				По умолчанию есть следующие реплики запуска:
					Запусти
					Погнали в
					Го в
				Если вы хотите добавить свою нажмите f3 и следуйте указаниям
				По умолчанию нет никаких приложений, поэтому добавьте их на f4
					Откроется окно выбора расположения приложения(возможно за окном консоли)
					Если у вас игра запускается через приложение, то укажите расположение exe файла игры
					Далее запишите реплику для игры без реплики запуска(Например, если я хочу добавить майнкрафт, то после указания его расположения я говорю майнкрафт(НЕ запути майнкрафт, а просто майнкрафт)
					Если запускается через ссылку(проверить можно зайдя в свойства ярлыка(это как правило игры из интернет магазинов по-типу steam, epic games), если будет ссылка, то игра запускается через ссылку)
					В таком случаее придётся поработать ручками:
						Укажите расположение левого файла
						Запишите реплику(ки)(аналогично с первым случаем)
						Сохраните
						Откройте папку cfg
						Откройте gm.txt
						Найдите путь к вашему "левому" файлу(оно будет в кавычках справа от %%%)
						Замените его на ссылку из ярлыка(кавычки оставьте)
						Сохраните файл
						Готово
			Настройка запуска сайтов
				Функция запуска сайта работает так:
					Говорите реплику для сайта
					Ассистент отвечает
					Открывается сайт
				По умолчанию есть YouTube, он запускается на реплики:
					Что посмотреть
					Открой YouTube
					Открой видосы
				Чтобы добавить сайт:
					Нажмите f5
					Вставьте заранее скопированную ссылку(Ctrl+V) или введите её
					Запишите реплику(Реплики для сайта не имеют реплики запуска, поэтому если например для YouTube нужно сказать не YouTube, а запусти YouTube)
					Сохраните
			Настройка активации голосового ввода:
				Функция голосового работает так:
					Говорите реплику голосового ввода
					Ассистент отвечает вам
					Вы говорите то,что хотели сказать
					После того как вы замолчите ассистент вам ответит
					В строке ввода появится сказанное вами(нужно чтобы на момент ответа ассистента строка ввода была активна)
					Так же сказанное вами будет скопировано в буфер обмена
				По умолчанию есть реплики:
					Голосовой ввод
					Ввод голосом
					Активируй голосовой ввод
					Активируй ввод голосом
					Активировать голосовой ввод
					Активировать ввод голосом
				Чтобы добавить реплику:
					Нажмите f6
					следуйте указаниям
			Настройка "Завершения работы":
				Функция завершения работы работает так:
					Вы говорите реплику завершения работы
					Ассистент отвечает вам
					Компьютер выключается
				По умолчанию есть реплики:
					Завершение работы
					Вырубай
					Закругляйся
				Чтобы добавить реплику:
					Нажмите f7
					следуйте указаниям
			Настройка web запроса:
				Функция запроса работает так:
					Вы говорите реплику запроса и сам запрос
					Ассистент отвечает
					Открывается вкладка браузера с вашим запросом(нужно чтобы в настройках был указан браузер по умолчанию)
				Реплики запроса следующие:
					Загугли
					Найди
					Браузер
			Настройка голосового запроса(экспирементальная функция):
				Функция голосового запросв работает так:
					Вы говорите реплику голосового запроса и запрос(запрос должен быть небольшим)
					Ассистент обрабатывает запрос(можент занять несколько секунд)(он переводит слова в начальную форму), находит первую статью на википедии, озвучивает первый абзац с неё(чем больше абзац и слабее пк тем дольше ждать)
					Для остановки нажмите любую клавишу на клавиатуре
				По умолчанию реплики:
					Расскажи о
					Расскажи об
					Расскажи про
					Назови
					Голосовой запрос
				Чтобы добавить реплику:
					Нажмите f8
					следуйте указаниям
			Настройка вызова плеера:
				Функция вызова плеера работает так:
					Вы говорите реплику вызова
					Ассистент отвечает
					Запускается плеер
				По умолчанию расположения плеера нет, поэтому его нужно добавить на f11
				Реплики вызова по умолчанию:
					Музыку
					Вруби музыку
					Включи музыку
					Поставь пластинку
				Чтобы добавить реплику вызова:
					Нажмите f9
					следуйте указаниям
			Настрока горечей клавиши:
				Чтобы запустить ассистента нужно нажать горячую клавишу(по умолчанию f10)
				Для её смены:
					Нажмите f12
					Следуйте указаниям
		Настройка завершена!
	Если вы хотите чтобы асистент запускался вместе с компьютером, то:
		Откройте папку с ассистентом
		ПКМ по приложению асистента
		Создать ярлык
		Нажмите Win+R
		Введите Shell:startup
		В открывшуюся папку переместите ярлык
	Для отключения асистента откройти диспетчер задач и снимите задачу с приложения ассистента
	Помните ассистент забирает часть мощностей процессора и оперативной памяти
	Если ассистент не смог разобрать вашу команду он "перезагружается"
	Ассистент - эксперементальный прототип, поэтому может не всегда работать нормально.
